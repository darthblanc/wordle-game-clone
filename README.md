# wordle-game-clone
A command line version of the New York Times Wordle game with solver   
To run the game, run the Main.java file  

Here is an image of the game:  
![alt text](https://github.com/darthblanc/wordle-game-clone/blob/main/readme-images/Screenshot%202025-02-16%20164749.png)

If you want to update the dictionary you could update the wordle.txt file.
